## Run
--Command Line
prerequisites
- .NET SDK
- SQL Server
- Node
- Angular CLI
**Steps**
1. Open directory source\Web\Frontend in command line and execute npm i.
2. Open directory source\Web in command line and execute dotnet run.
3. Open https://localhost:8090.
